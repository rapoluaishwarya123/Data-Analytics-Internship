## 📊 Task 1: Sales & Revenue Analysis Dashboard

### Description:
Built a dashboard to analyze sales and revenue data.

### Features:
- Imported data using CSV
- Calculated KPIs (Total Revenue, Quantity, Average)
- Visualized data using Line, Bar, and Pie charts
- Created a dashboard using Python (Matplotlib)

### Tools Used:
- Python
- Pandas
- Matplotlib
- Google Colab
